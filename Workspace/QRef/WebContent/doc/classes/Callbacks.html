    - (void) userAuthValidateCredentialCallback(err, isValid)

    Standard callback used by the UserAuth security routines to return the status of a credential validation operation.

    Parameters:

    • err (Error) — An error object if an error occurred or null otherwise.
    • isValid (Boolean) — An true/false value indicating if the credential was valid. True indicates a valid credential while false indicates failure or invalid credentials.

    - (void) userAuthLoginCallback(err, token, isValid)

    Standard callback used by the UserAuth#login method to return the status and token from an authentication operation.

    Parameters:

    • err (Error) — An error object if an error occurred or null otherwise.
    • token (String) — A valid token if successful otherwise null.
    • isValid (Boolean) — An true/false value indicating if the credential was valid. True indicates a valid credential while false indicates failure or invalid credentials.

    - (void) userAuthUserFromTokenCallback(err, user)

    Standard callback used by the UserAuth#userFromToken method to return the status and user from a lookup operation.

    Parameters:

    • err (Error) — An error object if an error occurred or null otherwise.
    • user (UserSchemaInternal) — The user associated with the token or null if failed.

    - (void) userAuthCreateAccountCallback(err, success, errorCode)

    Standard callback used by the UserAuth#createAccount method to return the status from a user creation operation.

    Parameters:

    • err (Error) — An error object if an error occurred or null otherwise.
    • success (Boolean) — True if the operation was successful, false otherwise.
    • errorCode (Number) — A numeric errorCode indicating why the request failed or zero otherwise.

    - (void) userAuthIsInRoleCallback(err, isInRole)

    Standard callback used by the UserAuth security routines to return the status of a role check operation.

    Parameters:

    • err (Error) — An error object if an error occurred or null otherwise.
    • isInRole (Boolean) — An true/false value indicating if the user has the specified role. True indicates that the user possesses the specified role.
