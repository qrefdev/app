    Overview

    Object sent as the body of an HTTP POST request to create a product.

    Author:

    • Nathan Klick

    • QRef 2012

    Property Summary

    (String) name

    (Required) A unique internal name for the product. Might possibly be a SKU or PN.

    (String) description

    (Optional) A product description which will be visible on the product details screen.

    (Boolean) isPublished

    (Required) A true/false value indicating whether the product is published for consumer user.

    (String) appleProductIdentifier

    (Optional) An apple product code for the corresponding item in the iTunes store.

    (String) androidProductIdentifier

    (Optional) A google product code for the corresponding item in the google play store.

    (Boolean) isAppleEnabled

    (Required) A true/false value indicating whether this product is available for apple devices.

    (Boolean) isAndroidEnabled

    (Required) A true/false value indicating whether this product is available for android devices.

    (Number) suggestedRetailPrice

    (Required) The suggested retail price for this product. Price may vary in iTunes and Google Play stores.

    (String) productCategory

    (Required) An enumeration indicating general product category. Valid values are ['aviation', 'marine', 'navigation'].

    (String) productType

    (Required) An enumeration indicating the type of product. Valid values are ['checklist', 'manual', 'guide'].

    (ObjectId) aircraftChecklist

    (Optional) An associated base checklist for aircraft products. This category represents the stock checklist that the user receives when purchasing the product.

    (Boolean) isSampleProduct

    (Required) A true/false value indicating whether this product is a sample product which is included with the application at no charge.

    (String) serialNumber

    (Optional) An string indicating which aircraft, marine, or navigation serial numbers are supported by this product.

    (ObjectId) manufacturer

    (Required) The manufacturer associated with this product.

    (ObjectId) model

    (Required) The model associated with this product.

    (String) coverImage

    (Optional) A server-based relative path to the cover artwork for this product. This path should be relative to the server root.

    (String) productIcon

    (Optional) A server-based relative path to the icon for this product. This path should be relative to the server root.
