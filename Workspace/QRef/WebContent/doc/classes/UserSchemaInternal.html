    Class: UserSchemaInternal Abstract

    Defined in: src/schema/UserSchema.coffee

    Overview

    This class is abstract.

    Schema representing an individual user object including their associated credentials, email address, and password recovery information.

    Examples:

    MongoDB Collection 

    db.users

    MongoDB Indexes 

    db.users.ensureIndex({ userName: 1 }, { unique: true })

    Property Summary

    (String) userName

    (Required) The username used to perform authentication. This should always be the user's email address.

    (String) passwordHash

    (Required) A SHA-512 HMAC representing the user's password.

    (String) passwordSalt

    (Required) A random SHA-512 hash used as a salt value in the password HMAC.

    (String) firstName

    (Optional) The first name of the user.

    (String) lastName

    (Optional) The last name (surname) of the user.

    (String) emailAddress

    (Required) The user's email address.

    (Array<ObjectId>) roles

    (Optional) The list of roles possessed by this user.

    (ObjectId) recoveryQuestion

    (Optional) The chosen recovery question for this user.

    (String) recoveryAnswer

    (Optional) An SHA-512 HMAC represting the user's answer to their recovery question.
